## 常见分录示例

投资方出资100万：

|    |                      |               |
|----|----------------------|--------------:|
| 借 | 资产：库存现金       | 1,000,000.00  |
| 贷 | 所有者权益：实收资本 | 1,000,000.00  |

从银行贷款100万：

|    |                      |               |
|----|----------------------|--------------:|
| 借 | 资产：库存现金       | 1,000,000.00  |
| 贷 | 负债：贷款           | 1,000,000.00  |

花3万元购买设备：

|    |                      |               |
|----|----------------------|--------------:|
| 借 |  资产：固定资产      |     30,000.00 |
| 贷 |  资产：库存现金      |     30,000.00 |


交押金5000元：

|    |                      |               |
|----|----------------------|--------------:|
| 借 |  资产：其他应收款    |      5,000.00 |
| 贷 |  资产：库存现金      |      5,000.00 |
